# 학생 성적 관리 프로그램

학생의 성적을 입·출력하고, 이름으로 검색, 학생 정보 수정, 학생 정보 삭제 및 학생 정보를 총점순 정렬할 수 있는 프로그램

## Environment
Codeblocks 20.3.0.0

GNU GCC Complier

## Descripton
1. 함수 목록

|기능|내용|
|------|---|
|학생 정보 자동 입력|테스트2|
|학생 성적 및 과목별 평균 출력|테스트2|
|학생 검색|테스트2|
|학생 정보 수동 삽입|테스트2|
|학생 성적 수정|테스트2|
|학생 정보 삭제|테스트2|
|총점 기준 학생 정렬|테스트2|
